# HealthStack

- Make sure to install node
- Install dependencies
<hr />
- To open Admin Dashboard
Open a new terminal
<br />
cd /admin 
<br />
npm isntall
<br />
npm start
<hr />
- To open main HealthStack website
Open another new terminal
<br />
cd /client
<br />
npm install
<br />
npm start
<hr />
- To run server
Open another new terminal
<br />
cd /server
<br />
npm install
<br />
npm start
